## 📝 Descripción
Este repositorio contiene el Trabajo Práctico Integrador para la materia Programación I de la UTN (Año 2024). Desarrollamos una suite de juegos de palabras interactivos en consola.

## 🕹️ Juegos Desarrollados

### 1. Palabras Encadenadas
El jugador debe formar palabras encadenadas, donde cada palabra empieza con la última letra de la palabra anterior.

### 2. Adivina la Capital
El jugador tiene que escribir la capital de un país dado.

### 3. Carrera de Palabras
El jugador compite para formar palabras lo más rápido posible con las letras dadas.
